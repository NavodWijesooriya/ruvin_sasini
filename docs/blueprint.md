# **App Name**: Aethel

## Core Features:

- Modern Event Showcase: An elegant, single-page scroll experience featuring the wedding timeline, location details with map integration, and the couple's story.
- Digital RSVP Portal: A streamlined form for guests to confirm attendance, select meal preferences, and list dietary restrictions.
- RSVP Data Persistence: Secure storage of all guest responses and attendee details in a centralized database for reliable retrieval.
- Organizer Dashboard: A private, authenticated admin panel to view the real-time guest list, export attendee data, and track headcount summaries.
- AI Guest Message Personalizer: A tool powered by generative AI that creates tailored thank-you messages or arrival instructions based on guest responses and specific groups.
- Mobile-Optimized Experience: A fully responsive design ensuring guests can view details and RSVP easily from any device or screen size.

## Style Guidelines:

- Primary color: Sophisticated Earth Brown (#604E3C) used for elegant typography and structural elements.
- Background color: Warm Paper (#FAF7F2), providing a soft, classic backdrop that avoids harsh white contrast.
- Accent color: Deep Blush (#A14548) for call-to-action buttons and subtle celebratory highlights.
- Font pairing: 'Playfair' (serif) for high-contrast, elegant headlines and 'PT Sans' (sans-serif) for readable body text and functional form elements.
- Minimalist line art icons for locations, dining choices, and schedules, keeping the aesthetic light and clean.
- Generous white space (breathing room) with asymmetrical element placement to create a bespoke, agency-portfolio feel.
- Smooth parallax scrolling for imagery and soft fade-in transitions for text sections as the user navigates the story.
